## Principi UI

UI significa interfaccia utente: cio' che una persona vede e usa nel sito.

La UI di Opal Signal deve essere:

- leggibile;
- ordinata;
- professionale;
- scura;
- diretta;
- non decorativa senza motivo.

## Principi UX

UX significa esperienza utente: quanto e' chiaro e semplice il percorso per chi visita il sito.

La UX di Opal Signal deve aiutare il visitatore a capire:

- cosa fa Opal Signal;
- per chi e' pensato;
- perche' e' diverso;
- quale azione compiere dopo.
