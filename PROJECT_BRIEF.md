# Project Brief

## Progetto

Opal Signal.

## Visione

Costruire una struttura AI-native italiana capace di aiutare professionisti a chiarire il proprio posizionamento, comunicare meglio online e creare sistemi digitali di acquisizione clienti.

## Target principale

- psicologi;
- psicoterapeuti;
- coach;
- nutrizionisti;
- fisioterapisti;
- professionisti del benessere e della relazione.

## Cosa vende Opal Signal

Opal Signal non vende post carini o siti vetrina generici.

Costruisce sistemi composti da:

- posizionamento;
- messaggio;
- sito;
- funnel;
- contenuti;
- pubblicita';
- automazioni leggere;
- acquisizione clienti.

## Obiettivo della Fase 0

Ricostruire una base tecnica e documentale ordinata dopo aver verificato che il repository GitHub contiene solo un README minimale.

## Cosa non si costruisce ora

- homepage definitiva;
- CRM;
- SaaS;
- automazioni complesse;
- template cliente;
- nuove skill o nuovi agenti.
