# Opal Signal

Opal Signal e' una struttura AI-native italiana pensata per costruire sistemi digitali di acquisizione clienti per professionisti del benessere e della relazione.

Questa repository contiene la base tecnica e documentale del progetto:

- sito ufficiale Opal Signal;
- workflow AI-native;
- design system iniziale;
- componenti base riutilizzabili;
- documentazione strategica per lavorare con Codex, GitHub e Vercel.

## Stato attuale

Fase 0: ricostruzione base.

In questa fase non viene costruita la homepage definitiva. Viene creata una base semplice, ordinata e verificabile.

## Stack

- Next.js
- TypeScript
- Tailwind
- App Router
- struttura `src/`
- alias `@/*`

## Comandi

```bash
npm install
npm run dev
npm run build
npm run lint
```
